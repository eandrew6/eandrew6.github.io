WURFL Capability: is_mobile
-an alias for is_wireless_device.
-true if a device is wireless.
-specifically a mobile phone or a tablet are considered as wireless devices, but a desktop PC or a laptop are not.
WURFL Capability: is_phone
-true if device is a mobile phone.
WURFL Capability: is_tablet
-true if a device is a tablet computer (iPad and similar, regardless of OS).
