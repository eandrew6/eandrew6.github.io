SVG Files in CNIT 131A

Shapes

Rectangle

The 'rect' element, of SVG, is used to create rectangle shapes or variations of rectangle shapes. The width and the height of the rectangle shapes are defined by the attributes 'width' and 'height' and the 'style' attribute is used to define style properties. The CSS is used to define the fill colors, the stroke-width property for the width of the borders, and the stroke property for the color of the borders.

Circle

The 'circle' element, of SVG, is used to create circle shapes. The 'cx' and 'cy' attributes explain the x and y coordinates of the circles center and the 'r' attribute defines the circle's radius. The circles center is set to (0,0) when 'cx' and 'cy' are omitted.

Gradients

The 'gradient' element, of SVG, is used to transition one color to another and several colors can be applied at once. The two main types of gradients are 'linear' and 'radial' of SVG. The 'linearGradient' element is used to define a unique name for the gradient. The start and end of the positions of the gradient are defined by attributes 'x1', 'x2', 'y1', and 'y2' of the 'linearGradient' element. The color range can be composed of two or more colors for the gradient and each color is specified with a 'stop' element. The 'offset' element used to define the colors beginning and end and the ellipse element is linked to the gradient by the 'fill' attribute.
