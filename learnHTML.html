HTML Tags: Z-A

Meta Tags

Meta tags are used within HTML to describe the web page. According to references, the < meta > tag provides "metadata" about the HTML document. Meta data is not displayed on the page, but can be viewed by browsers and other web services. The content of this tag is used by some search engines when it lists the page in the search results. Remember that the meta tag always goes in the < head > element and has no ending tag. The following is an example of meta tags used within HTML:

<!DOCTYPE html>
<html lang = "en">
<head>
<title> My first web page </title>
<meta name = "description" content = "This is my first page. The topic of discussion is HTML Tags...">
<meta charset = "utf-8">
</head>"

<body>
...
</body>

</html>

The "name" and "content" words mentioned within the meta tag are called attributes and are required. The meta tag "<meta charset = "utf-8">" is used to specify the character encoding of the content. According to reference, most web pages will use this character encoding.

Header Tags

Header tags are used to surround the content that will be apart of a headline. Typically located at the top of the web page, the header tag <header> is placed between the HTML tag <html> and the Body tag <body> within the web page content. According to reference, the idea of the tag is to bring document structure and semantics to the arena of web development. Note that more than one header tag can be used within your HTML document. The following includes examples of using the header tag <header> and the <h1>, <h2>, etc...tag:

<!DOCTYPE html>
<html lang = "en">

<head>
<title> My first web page </title>
<meta name = "description" content ="This is my first page...">
<meta charset = "utf-8">
</head>

<body>

<header> ... </header>

<h1> Header Tag </h1>
<h2> Another Header Tag </h2>
<h3> And...Another Header Tag </h3>
<h...

</body>
</html>

Headings are defined with the <h1> to <h6> tags, <h1> being the most important and <h6> being the least important. Note: <h1> to <h6> will determine how large to how small the font placed inside the tags will appear.
